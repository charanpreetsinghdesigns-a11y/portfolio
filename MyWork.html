Design Process
Research
Research includes understanding about the project scope, performing competitive analysis as well as doing the primary and secondary user research.
Requirements
After all the research and analysis part, problem and vision statements are introduced. This step also involves the User personas and User Scenarios.
Framework
Framework is the base of the design which includes the work on backend structure. This step involves creating the Information Architecture and User flows
Detailed Design
This is the last step in the design process that focus on creating color palettes, introducing typography, wireframing UI screens and working on prototyping
