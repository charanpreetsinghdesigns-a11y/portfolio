Upsurge

Kids' Based Edutainment Platform

May – Dec 2023 #WebsiteRedesign #AppDesign #Edtech
About Upsurge

Upsurge is a family-focused financial platform that offers financial education for children through interactive educational content, games, and gamified real-life responsibility management.

Challenge — Website Redesign

In the first 4 months, I focused on improving the upsurge website. I worked on both the mobile and desktop versions, studying the design to make the website easier and more enjoyable to use.

Website Design Goals
  • Reduce scroll time for landing page (Mobile version)
  • Revisit pre-signup web elements
  • Redesign Signup Experience
  • Redesign Post-sign in Experience (Dashboard)
Process — Website
  • Research & Analysis: Based on previous analysis about scroll time and user retention, I started working on landing page elements. Screenshots of current website interface were collected, all components were designed exactly the same and the current design was studied.
  • Heuristic Evaluation: While redesigning, I used laws of heuristic evaluation on the elements of the website to understand about the drawbacks in current design.
  • Information Architecture: Based on the research findings, I restructured website's navigation and content, prioritizing features and information according to user needs.
Challenge — Mobile App Redesign

In the last 4 months, I focused on the Mobile Application which was not designed properly and not prioritized in the past. All the elements were outdated and not aligned with the platforms' values.

Mobile App Goals
  • Redesign whole experience
  • Add gamification elements
  • Make it kids' friendly
Process — Mobile
  • Colors and Typography: Since upsurge is a kids-based edtech platform, I worked on colors and typography to make it more vibrant and clean at the same time. I chose a yellowish shade to make it more enjoyable for kids.
  • Gamification: I also tried to add possible gamification elements to encourage kids to spend more time and continue learning while getting rewards.
Conclusion

My 8-month journey at upsurge has been a continuous learning experience. From website designs to crafting gamified mobile experiences, I've expanded my skills and knowledge in UX design. As I reflect on this journey, I'm not just closing a chapter but carrying forward a wealth of insights and lessons that will undoubtedly shape my future endeavors in UX design.
